# Homebrew Tap for Tunnee

This repository contains the Homebrew formula for the [Tunnee CLI](https://tunnee.com).

Tunnee helps you expose local services to the internet for webhook testing, mobile app development, OAuth callbacks, and other local-to-public workflows.

## Install

```bash
brew install tunnee-dev/tap/tunnee
```

## Upgrade

```bash
brew upgrade tunnee
```

## Uninstall

```bash
brew uninstall tunnee
```

## Notes

- This tap is maintained for the Tunnee CLI.
- Formula updates are published automatically from Tunnee releases.
- If you already tapped this repository manually, you can continue using the same install command.
